# 斯蒂尔切斯积分

## 核心内容
1. **黎曼-斯蒂尔切斯积分**：设 f 定义在 [a,b] 上，alpha 是 [a,b] 上的单调函数，定义和式
   Σ(i=1 到 n) f(xi_i) * (alpha(x_i)-alpha(x_{i-1}))
   若极限存在，则称 f 关于 alpha 在 [a,b] 上黎曼-斯蒂尔切斯可积。
2. 可积条件：当 f 连续、alpha 有界变差时，积分存在。
3. 与勒贝格积分的关系：当 alpha(x)=x 时，黎曼-斯蒂尔切斯积分退化为黎曼积分。
4. 分部积分公式：若 f 关于 alpha 可积，则 alpha 关于 f 可积，且
   从 a 到 b 的积分 f d(alpha) + 从 a 到 b 的积分 alpha df = f(b)alpha(b)-f(a)alpha(a)。
