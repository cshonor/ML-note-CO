# 3.6 非平稳数据的差分模型（ARIMA）

## 核心问题：非平稳序列
许多实务序列（如价格、指数水平）呈现趋势或单位根行为，均值或二阶结构随时间变化，不宜直接套平稳 ARMA。

---

## 差分：把非平稳转成近似平稳
对序列做差分以削弱趋势或随机游走成分：

- **一阶差分**：nabla x_t = x_t - x_{t-1}
- **d 阶差分**：重复差分 d 次，记为 nabla^d x_t（记号与教材一致即可）

---

## ARIMA(p,d,q) 模型形式
先对原序列做 d 阶差分得到（近似）平稳序列 y_t，再对 y_t 建 ARMA(p,q)：

nabla^d x_t 服从 ARMA(p,q)

即常见写法：**ARIMA = 差分 + ARMA**。

---

## 如何确定差分阶数 d
1. 时序图：是否有明显趋势、是否像随机游走  
2. **ADF 单位根检验**：  
   - p 值较大时，证据不足拒绝单位根，常需差分  
   - 差分后再次检验，直到 p 值足够小（与显著性水平比较）

注意：过差分会引入不必要的移动平均结构，d 并非越大越好。

---

## Python 示例
```python
import pandas as pd
from statsmodels.tsa.stattools import adfuller

result = adfuller(x.dropna())
print("ADF 统计量:", result[0], "p 值:", result[1])

x_diff = pd.Series(x).diff().dropna()
result_diff = adfuller(x_diff)
print("一阶差分后 p 值:", result_diff[1])
```
