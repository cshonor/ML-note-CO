# 基（Basis）—— 向量空间的最小坐标系

## 1. 基的严格定义
设 V 是数域 F 上的向量空间，向量组 B = (v1, v2, ..., vn) 包含于 V。  
B 称为 V 的一组基，当且仅当同时满足：
1. 线性无关：k1*v1 + ... + kn*vn = 0 只能推出 k1 = ... = kn = 0
2. 张成全空间：span(v1, ..., vn) = V

简言之：基 = 线性无关的生成集。

## 2. 基的核心本质
### 2.1 基是最精简坐标系
- 无冗余：没有一个向量能由其余向量线性表示
- 刚好够用：空间中任意向量都能由这组向量线性组合得到

### 2.2 唯一表示定理（基的灵魂）
若 B = (v1, ..., vn) 是 V 的一组基，则对任意 v 属于 V：
- 存在一组系数 lambda1, ..., lambdan 使 v = lambda1*v1 + ... + lambdan*vn
- 且这组系数唯一

这些系数 (lambda1, ..., lambdan) 叫做 v 在基 B 下的坐标。

唯一性证明（标准写法）：
若 v = sum(lambda_i*vi) = sum(mu_i*vi)，两式相减得
sum((lambda_i - mu_i)*vi) = 0。  
由 B 线性无关，得到 lambda_i - mu_i = 0，即 lambda_i = mu_i，故表示唯一。

## 3. 两大基构造定理（必考）
### 3.1 基扩充定理（无关 -> 基）
在有限维空间中，任意一组线性无关向量都可以补充若干向量，扩成全空间的一组基。

几何直观：
- 在 R^3 中，一条过原点的方向向量可扩成平面基，再扩成三维基。

### 3.2 基缩减定理（张成 -> 基）
在有限维空间中，任意一组张成全空间的向量都可以删去冗余向量，缩成一组基。

几何直观：
- 在 R^3 中若给了 4 个张成向量，必有冗余，可删到 3 个，仍张成且无关。

## 4. 子空间的基
设 U 是 V 的子空间。在有限维条件下：
1. U 一定存在自己的基
2. 若 dim(V) = n, dim(U) = m，则 m <= n

例子：
R^3 中任意过原点平面的维数是 2，因此它的基通常含 2 个无关向量，且 2 <= 3。

## 5. 重要推论（考场秒杀）
1. 同一空间任意两组基的向量个数相同（这个个数就是维数）
2. 在 n 维空间中：
   - 任意 n 个线性无关向量必构成一组基
   - 任意 n 个能张成全空间的向量必构成一组基

## 6. 一句话口诀
- 基 = 无关又张成
- 唯一表示是灵魂
- 无关可扩，张成可缩
- 子空间维数不超过全空间
