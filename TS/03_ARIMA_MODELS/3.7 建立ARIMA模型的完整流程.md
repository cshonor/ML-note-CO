# 3.7 建立 ARIMA 模型的完整流程

## 标准建模步骤（Box–Jenkins 方法）

### 1. 序列预处理与平稳性检验
- 画时序图，观察趋势、季节、异常点  
- ADF 等检验判断是否需差分，确定 d

### 2. 模型识别（定阶 p、q）
- 对近似平稳序列画 ACF/PACF，做粗筛  
- 在合理网格上搜索 (p,q)，用 AIC/BIC 比较

### 3. 参数估计
- 以 MLE 为主流输出参数与标准误

### 4. 模型诊断
- 残差是否近似白噪声（Ljung–Box、残差 ACF）  
- 若残差仍有结构，调整阶数或考虑季节项、异方差等

### 5. 预测与评估
- 样本外或滚动评估 MAE、RMSE 等

---

## Python 流程示例（骨架）
```python
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.stattools import adfuller
from statsmodels.tsa.arima.model import ARIMA
from statsmodels.stats.diagnostic import acorr_ljungbox

# 1) 读入数据（按你的数据源替换）
# data = pd.read_csv("time_series.csv", index_col="date", parse_dates=True)
# x = data["value"].astype(float)

# 2) 差分与 ADF（示例：对 x 一阶差分）
x_diff = x.diff().dropna()
print("差分后 ADF p 值:", adfuller(x_diff)[1])

# 3) 拟合 ARIMA(p,d,q)（示例 order=(1,1,1)）
model = ARIMA(x, order=(1, 1, 1))
res = model.fit()
print(res.summary())

# 4) 残差诊断
residuals = res.resid.dropna()
lb = acorr_ljungbox(residuals, lags=[10], return_df=True)
print("Ljung-Box 检验:\n", lb)

# 5) 预测
fc = res.get_forecast(steps=10)
print(fc.predicted_mean)
print(fc.conf_int())
```

说明：`Ljung-Box` 的 p 值若**不**显著小，常表示尚无充分证据拒绝「残差为白噪声」；具体阈值与滞后集合需结合研究设计。
