# 外测度

## 1. 从第二章无缝衔接到第三章

第二章我们研究的是“集合长什么样”：内点、聚点、开集、闭集、完备集、康托尔集。  
第三章开始研究“集合有多大”：给一般点集定义长度、面积、体积。

区间 [a,b] 的长度是 b-a，这很熟悉。  
但像有理数集、康托尔集、散点集，不是单个区间，怎么定义长度？  
答案就是：先用外测度。

## 2. 外测度的定义逻辑

核心思想：从外面用可数个开区间把集合 E 盖住。

步骤：
1. 任取一组开区间 I1, I2, ...，满足 E 包含于并 I_k
2. 计算覆盖总代价：sum(|I_k|)
3. 在所有可能覆盖里取下确界（最小可能总长）

定义：
m*(E) = inf { sum(|I_k|) | E 包含于并 I_k，且 I_k 为开区间 }

一句话：外测度就是“从外面盖住 E 所需的最小总长度”。

## 3. 最熟例子：E = (0,1)

覆盖方式举例：
- 用 (-0.1,1.1) 盖住，长度 1.2
- 用 (0,1) 盖住，长度 1

显然可以做到总长 1，且不可能小于 1，所以：
m*((0,1)) = 1。

## 4. 端点与边界为什么不影响外测度

比较 E1 = (0,1) 与 E2 = [0,1]：
- 两者外测度都等于 1

原因：多出来的端点 0、1 只是有限个点。  
单点可用任意小区间覆盖，代价可做得任意接近 0。  
所以“加减有限点”通常不改变外测度。

这和你前面关于边界点、聚点的直觉可以兼容：
边界点在拓扑上很关键，但在长度上可能贡献为 0。

## 5. 外测度四条核心性质

1. 非负性：m*(E) >= 0，且 m*(空集) = 0
2. 单调性：A 包含于 B，则 m*(A) <= m*(B)
3. 次可数可加性：m*(并 E_k) <= sum(m*(E_k))
4. 平移不变性：对任意 a，有 m*(E+a) = m*(E)

## 6. 为什么先学外测度

因为第三章后续的“可测集”就是在外测度基础上筛选出来的。  
课程逻辑是：
点集拓扑（开闭聚点） -> 一维开集构造 -> 用开区间覆盖 -> 外测度 -> 可测集 -> 积分。

外测度是测度论的入口。
