# 3.8 使用自相关误差进行回归（ARMAX 与相关）

## 问题：普通线性回归的缺陷
当回归误差项存在**自相关**（不是白噪声）时，OLS 系数仍可能无偏，但：

- 高斯–马尔可夫意义下的「最优」性不再保证  
- 常用标准误公式往往**偏误**，t 检验、置信区间可能失真

---

## 模型形式：自相关误差（回归 + ARMA 误差）
观测方程：

y_t = beta_0 + beta_1 * x_{1t} + ... + beta_k * x_{kt} + eta_t

其中 eta_t 本身满足 ARMA(p,q)：

eta_t = phi_1 * eta_{t-1} + ... + phi_p * eta_{t-p} + w_t + theta_1 * w_{t-1} + ... + theta_q * w_{t-q}

这就是**带 ARMA 误差的回归**；与「动态回归 / ARMAX」等名称在文献中有细微差别，核心都是：**均值方程 + 相关误差结构**。

---

## 解决思路（概念）
1. 先做 OLS 得残差，检查残差 ACF  
2. 对残差拟合 ARMA 或联合用 GLS / MLE 同时估计回归与误差过程（实务以后者为主）

---

## 与含外生变量的 ARMA 的关系
若在均值方程中加入外生解释变量，并在噪声部分保留 MA/AR 结构，常见在软件中通过 **ARIMA 的 exog** 或 **SARIMAX** 等接口实现，例如：

y_t =（外生项线性组合）+（ARMA 噪声）

具体语法以 statsmodels 版本文档为准。
