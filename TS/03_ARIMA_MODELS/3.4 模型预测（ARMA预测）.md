# 3.4 模型预测（ARMA 预测）

## 预测的核心思想
用过去的观测值与误差项结构，在线性框架下外推未来取值。

---

## 三种预测形式（概念）

### 1. 条件期望预测（最小均方误差思路）
对未来 h 步，记预测为 x_hat_{t+h|t}，直观上是在已知信息集下的条件期望：

x_hat_{t+h|t} = E( x_{t+h} | x_t, x_{t-1}, ... )

- h=1（一步预测）：常可直接代模型递推
- h>1（多步预测）：未来观测未知，需用**已有预测值**递推代入，不确定性随 h 累积

### 2. AR(p) 预测
线性部分只依赖过去预测值：

x_hat_{t+h|t} = phi_1 * x_hat_{t+h-1|t} + ... + phi_p * x_hat_{t+h-p|t}

（式中若下标对应真实已观测期，则用真实值；否则用预测值。）

当 h 足够大时，纯 AR 的预测常逐步向序列均值（或确定性部分）靠拢。

### 3. MA(q) 预测
MA 依赖有限个未来不可见冲击；当 h>q 时，可预报部分往往退化为常数均值，预测误差方差趋于过程无条件方差（具体公式依参数而定）。

---

## 预测区间
在近似正态等假设下，可对 x_hat_{t+h|t} 给出置信区间，反映 h 步预测的不确定性（多步通常更宽）。

---

## Python 示例（ARIMA 接口表示 ARMA）
statsmodels 里差分阶数 d=0 时，ARIMA(p,0,q) 即 ARMA(p,q)。

```python
from statsmodels.tsa.arima.model import ARIMA

# x 为一维序列；拟合 ARMA(1,1) 即 order=(1, 0, 1)
model = ARIMA(x, order=(1, 0, 1))
res = model.fit()

forecast = res.get_forecast(steps=10)
mean_forecast = forecast.predicted_mean
conf_int = forecast.conf_int()  # 近似置信区间
print(mean_forecast)
print(conf_int)
```
