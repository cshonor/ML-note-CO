# §1.5 事件的独立性

## 一、两个事件独立的定义
若满足：
P(AB) = P(A) * P(B)
则称 **事件A与事件B相互独立**。

### 等价条件（P(B)>0）
P(A|B) = P(A)

### 核心白话
B发不发生，**完全不影响A的概率**；两者互不干扰、互不提供信息。

### 简单例子
1. 两次独立抛硬币：
   第一次正面、第二次正面，互相独立。
2. 掷骰子和抛硬币，两个试验完全独立。

## 二、独立 与 互斥 严格区分（极易混淆考点）
1. **互斥**：AB = empty set，不能同时发生，是**集合关系**
2. **独立**：P(AB)=P(A)P(B)，概率互不影响，是**概率关系**

> 独立 ≠ 互斥；互斥 ≠ 独立，没有必然推导关系。

## 三、独立事件的推论性质（两事件）
若 A、B **相互独立**（即 P(AB) = P(A)P(B)），则下面三组也全都独立：

1. A 与 非B  
2. 非A 与 B  
3. 非A 与 非B  

**一句话记**：独立事件，**取对立（补）仍然独立**。

### 简要证明（以 A 与 非B 为例）
由 A 可拆成两块不交并：A = AB 并 A(非B)，故

P(A) = P(AB) + P(A 非B)，即 P(A 非B) = P(A) - P(AB)。

独立时 P(AB) = P(A)P(B)，代入得

P(A 非B) = P(A) - P(A)P(B) = P(A)(1 - P(B)) = P(A)P(非B)。

满足独立定义 P(A 非B) = P(A)P(非B)，故 A 与 非B 独立。**非A 与 B**、**非A 与 非B** 同理可证（对称地拆 B 或拆 A）。

## 四、多个事件的独立性

### 1. 两两独立
对 n 个事件 A1, A2, …, An，**任意两个**不同下标 i、j 都满足：

P(AiAj) = P(Ai)P(Aj)（i 不等于 j）

含义：**任意两个之间**在概率上互不影响。

### 2. 相互独立（更强）
不仅两两独立，还要求：**任意取若干个事件求交**，其概率都等于各事件概率的乘积（教材对有限个事件会给出等价的一组等式）。

以 **三个事件** A, B, C 为例，通常要同时验证下面 **4 条**（前 3 条即两两独立，第 4 条把三者“绑在一起”）：

- P(AB) = P(A)P(B)  
- P(AC) = P(A)P(C)  
- P(BC) = P(B)P(C)  
- P(ABC) = P(A)P(B)P(C)  

### 关键关系（必背）
- **相互独立 ⇒ 两两独立**（一定成立）  
- **两两独立 推不出 相互独立**（反例存在，见下）

### 经典反例（两两独立但不相互独立）
抛两枚**均匀**硬币，样本空间四个结果等概率：{正正, 正反, 反正, 反反}。设

- A：第一枚为正面  
- B：第二枚为正面  
- C：两枚结果相同（正正或反反）  

则 P(A) = P(B) = P(C) = 1/2。可算得

- P(AB) = P(正正) = 1/4 = P(A)P(B)，同理 P(AC) = P(BC) = 1/4，**两两独立**；  
- 但 ABC 表示“第一正、第二正、且同面”，即正正，故 P(ABC) = 1/4，而 P(A)P(B)P(C) = 1/8，**不相等**，故 **不相互独立**。

## 五、独立重复试验
### 定义
同一试验重复多次，各次试验**相互独立**，且每次试验概率结构完全不变。

### 典型模型
- 伯努利试验：每次只有“成功/失败”两种结果，独立重复
- 二项分布、几何分布、伯努利概型，全部建立在**独立试验**基础上

### 生活化例子
有放回摸球、多次独立射击、重复做题蒙答案，都是独立试验。
