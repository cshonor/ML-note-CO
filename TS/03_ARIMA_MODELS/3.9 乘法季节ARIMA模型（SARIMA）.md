# 3.9 乘法季节 ARIMA 模型（SARIMA）

## 问题：带季节性的时间序列
月度销量、季度用电等常有**固定周期**的重复形态，仅用非季节 ARIMA 往往不够。

---

## SARIMA(p,d,q)(P,D,Q)_s 记号
- **p,d,q**：非季节部分的 AR 阶、差分阶、MA 阶  
- **P,D,Q**：季节部分的 AR 阶、季节差分阶、MA 阶  
- **s**：季节周期（例如月数据常取 s=12，季数据常取 s=4）

---

## 模型形式（文字版）
用滞后算子 B（B x_t = x_{t-1}）写时，常见写法为：

非季节因子 phi(B) 与季节因子 Phi(B^s) 作用在适当差分后的序列上，

右侧为季节与非季节 MA 因子 theta(B)、Theta(B^s) 乘以白噪声 w_t。

季节差分算子常记为：

nabla_s = 1 - B^s

（与「先消除季节周期、再消除趋势」的实务步骤对应。）

---

## 建模关键步骤
1. 由业务或谱图、ACF 在 s,2s,... 处的峰值判断 **s**  
2. 做 **D 阶季节差分** 弱化季节随机游走  
3. 再做 **d 阶普通差分** 弱化趋势  
4. **定阶**：  
   - (p,q) 看差分后序列在**非季节滞后**上的 ACF/PACF  
   - (P,Q) 关注滞后 s、2s、... 附近的尖峰与截断形态

---

## Python 示例（SARIMAX）
```python
from statsmodels.tsa.statespace.sarimax import SARIMAX

# 月度数据示例：SARIMA(1,1,1)(1,1,1,12)
model = SARIMAX(x, order=(1, 1, 1), seasonal_order=(1, 1, 1, 12))
res = model.fit()
print(res.summary())

forecast = res.get_forecast(steps=12)
print(forecast.predicted_mean)
```
