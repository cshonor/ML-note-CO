# §2.6 随机变量函数的分布（完整版，含例题）

已知 X 的分布，求 **Y = g(X)** 的分布。离散与连续套路不同；公式一律**纯文本**。

---

## 2.6.1 离散随机变量函数的分布

### 核心方法
**直接映射 + 合并同值概率。**

已知 X 的分布列 P(X=x_k)=p_k，求 Y=g(X)：

1. 对每个 x_k，算出 y_k = g(x_k)；  
2. 若有多个 x 映到同一个 y，把这些 x 的概率**相加**；  
3. 列出 Y 的全部取值及对应概率，检查概率和为 1。

### 例 1
X ~ B(1, 0.5)，即 P(X=0)=0.5，P(X=1)=0.5。设 Y = 2X + 1。

| X | 0 | 1 |
| :---: | :---: | :---: |
| Y = 2X+1 | 1 | 3 |
| P | 0.5 | 0.5 |

故 Y 只取 1、3，各 0.5。

### 例 2
X 取值 -1, 0, 1, 2，概率依次为 0.2, 0.3, 0.1, 0.4。设 Y = X^2。

- Y 的可能取值：0, 1, 4。  
- P(Y=0) = P(X=0) = 0.3。  
- P(Y=1) = P(X=-1) + P(X=1) = 0.2 + 0.1 = 0.3。  
- P(Y=4) = P(X=2) = 0.4。

---

## 2.6.2 连续随机变量函数的分布

设 X 的密度为 f_X(x)，Y = g(X)。

### 方法一：分布函数法（通用，不要求单调）

1. 先求 Y 的分布函数：  
   F_Y(y) = P(Y <= y) = P(g(X) <= y)。  
   把事件 {g(X) <= y} 化成 X 落在某个集合上，再对 f_X 在该集合上积分：  
   F_Y(y) = 在集合 {x : g(x) <= y} 上对 f_X(x) dx 积分。

2. 在 F_Y 可导处，**f_Y(y) = F_Y'(y)**（分段求导时注意接头）。

### 例（分布函数法）
X ~ U(0,1)，f_X(x)=1（当 0<x<1），否则 0。Y = 2X + 1。

当 1 < y < 3 时，2X+1 <= y 等价于 X <= (y-1)/2，且 (y-1)/2 落在 (0,1) 内，故

F_Y(y) = P(X <= (y-1)/2) = 从 0 到 (y-1)/2 对 1 dx 积分 = (y-1)/2。

求导得 f_Y(y) = 1/2（当 1<y<3）；区间外密度为 0。即 Y ~ U(1,3)。

---

### 方法二：公式法（严格单调、可导、且 g'(x) 不变号）

设 Y = g(X)，g 在 X 的支撑上**严格单调**、可导且 g'(x) 不为 0。反函数记为 x = h(y)，则

f_Y(y) = f_X( h(y) ) * |h'(y)|，

y 落在 g(x) 的值域（开区间 min(g) 到 max(g)）内；其余 y 处常取 f_Y(y)=0。

### 例（公式法，同上题）
Y = 2X+1，反函数 x = h(y) = (y-1)/2，h'(y) = 1/2。当 1<y<3 时 h(y) 在 (0,1) 内，f_X(h(y))=1，故

f_Y(y) = 1 * |1/2| = 1/2，

与分布函数法一致。

---

### 非单调函数（如 Y = X^2）

g 分段单调时，可对每一段用公式法，再把**同一点 y 上来自各段的贡献相加**：

f_Y(y) = 对所有满足 y = g(x) 的单调分支 i，求和：f_X( h_i(y) ) * |h_i'(y)|，

其中 h_i(y) 为第 i 段上的反函数（具体区间要按 X 的支撑划分）。**稳妥做法**仍是先写 F_Y(y) 再分段求导。

---

## 小结（解题思路）

| 情形 | 做法 |
| :--- | :--- |
| **离散** | 算 y=g(x)，**合并**映到同一 y 的概率。 |
| **连续且单调** | 优先 **f_Y = f_X(h(y)) * abs(h'(y))**；或分布函数法核对。 |
| **连续非单调** | **分布函数法**最稳；或分段反函数叠加密度。 |
