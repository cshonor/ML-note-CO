# §3.1 多维随机变量及其联合分布

## 1. 二维随机变量定义
设 X, Y 是同一样本空间上的两个随机变量，称 (X, Y) 为二维随机变量。

## 2. 联合分布函数
F(x,y) = P(X<=x, Y<=y)，对任意实数 x,y。

性质：
1. 对 x,y 单调不减、右连续
2. F(-∞,-∞)=0，F(+∞,+∞)=1
3. 矩形概率：
   P(a<X<=b, c<Y<=d)
   = F(b,d)-F(b,c)-F(a,d)+F(a,c)

## 3. 二维离散型联合分布列
p_ij = P(X=x_i, Y=y_j)

满足：
sum_i sum_j p_ij = 1，且 p_ij >= 0

## 4. 二维连续型联合密度
F(x,y) = 从 -∞ 到 x、再从 -∞ 到 y 的二重积分 f(u,v)dudv

条件：
f(x,y)>=0，且全平面二重积分 f(x,y)dxdy = 1
