# 不定积分

## 核心内容
1. **绝对连续函数**：定义在 [a,b] 上的函数 f，若对任意 epsilon>0，存在 delta>0，当互不相交的区间列长度和小于 delta 时，对应函数值差的和小于 epsilon，则称 f 绝对连续。
2. **勒贝格积分的牛顿-莱布尼茨公式**：若 f 在 [a,b] 上勒贝格可积，则其不定积分
   F(x) = 从 a 到 x 的积分 f(t)dt
   是绝对连续函数，且几乎处处可导，导数等于 f(x)。
3. 绝对连续函数的性质：几乎处处可微，且导数勒贝格可积；是有界变差函数的子类。
4. 逆定理：若 F 在 [a,b] 上绝对连续，则 F' 勒贝格可积，且
   F(x)-F(a) = 从 a 到 x 的积分 F'(t)dt。
