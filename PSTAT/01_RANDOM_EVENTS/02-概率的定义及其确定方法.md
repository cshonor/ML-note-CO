# §1.2 概率的定义及其确定方法

## 一、概率的直观理解
概率是用来**度量随机事件发生可能性大小**的数值。
- 取值范围：0 <= P(A) <= 1
- P(A)越接近1：越容易发生
- P(A)越接近0：越难发生

## 二、概率的公理化定义（严谨数学基石）
给定样本空间 Omega、事件域 F，对任意事件 A 属于 F，定义集合函数 P(A)，满足三条公理：

1. **非负性**
P(A) >= 0

2. **规范性**
P(Omega) = 1

3. **可列可加性**
若 A1, A2, ... 两两互不相容，则：
P(A1并A2并...) = P(A1) + P(A2) + ...

> 后面所有概率公式、性质，全部由这三条公理严格推导而来。

## 三、古典概型（等可能概型）
### 1. 适用两个条件
1. 样本空间**样本点有限**
2. 每一个基本结果**发生可能性完全相等**

### 2. 计算公式
P(A) = 事件A包含的样本点个数 / 样本空间总样本点个数

### 3. 经典例题
掷一枚均匀骰子，求“点数为偶数”的概率。
- 总样本点：6 个
- A={2,4,6}，包含3个样本点

P(A) = 3/6 = 1/2 = 0.5

## 四、几何概型
### 1. 适用条件
1. 样本空间是连续几何区域：长度、面积、体积
2. 区域内任意点**均匀等可能**

### 2. 计算公式
P(A) = 事件A对应几何度量 / 样本空间总几何度量

### 3. 经典例子
在区间 [0,2] 上任取一点，求落在 [0.5,1.5] 的概率。
- 总长度：2-0=2
- 有效长度：1.5-0.5=1

P = 1/2 = 0.5

## 五、频率确定概率（统计概率）
### 1. 频率定义
重复试验 n 次，事件A发生 n_A 次：
f_n(A) = n_A / n

### 2. 频率稳定性
当试验次数 n **充分大**时，频率 f_n(A) 会在某个固定数值附近摆动，这个稳定值就是**概率 P(A)**。

### 3. 例子
抛硬币上万次，正面出现频率会无限趋近 0.5。
> 这也是大数定律的直观雏形。

## 六、主观概率
不是靠试验、不是靠几何，而是依托：经验、信息、专业判断，人为给出的置信概率。
- 应用场景：金融风控、投资预判、贝叶斯先验、天气预判、专家决策
- 例子：分析师判断“明年通胀上行概率70%”，就是主观概率。
