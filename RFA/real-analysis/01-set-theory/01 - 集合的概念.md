# 实变函数 第一章 第一节：集合

## 一、集合与元素

### 1. 基本定义

具有某种确定性质的对象全体称为集合，集合中的对象称为元素。

集合的三个基本特征：

- 确定性：任一对象要么属于集合，要么不属于
- 互异性：集合中元素互不重复
- 无序性：元素次序不影响集合本身

### 2. 表示方法

- 列举法：`A = {a1, a2, ...}`
- 描述法：`A = {x | x 满足性质 P}`

常用数集记号：

- `N`：自然数集
- `Z`：整数集
- `Q`：有理数集
- `R`：实数集
- `∅`：空集

## 二、集合关系

- 属于：`x ∈ A`
- 不属于：`x ∉ A`
- 子集：`A ⊂ B`
- 真子集：`A ⊊ B`
- 相等：`A = B` 当且仅当 `A ⊂ B` 且 `B ⊂ A`

规定：空集是任意集合的子集，即 `∅ ⊂ A`。

## 三、集合运算

### 1. 并集

`A ∪ B = {x | x ∈ A 或 x ∈ B}`

### 2. 交集

`A ∩ B = {x | x ∈ A 且 x ∈ B}`

若 `A ∩ B = ∅`，称 `A` 与 `B` 互不相交。

### 3. 差集

`A \ B = {x | x ∈ A 且 x ∉ B}`

### 4. 补集（相对全集 `X`）

`A^c = X \ A`

## 四、运算律（必背）

- 交换律：`A ∪ B = B ∪ A`，`A ∩ B = B ∩ A`
- 结合律：`(A ∪ B) ∪ C = A ∪ (B ∪ C)`，`(A ∩ B) ∩ C = A ∩ (B ∩ C)`
- 分配律：`A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)`，`A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)`
- 德摩根公式：`(A ∪ B)^c = A^c ∩ B^c`，`(A ∩ B)^c = A^c ∪ B^c`

可推广到可数或任意指标族的并、交。

## 五、集合列极限（入门版）

设 `A_n` 为集合列：

- 上极限：
  `limsup A_n = {x | x 属于无穷多个 A_n}`
- 下极限：
  `liminf A_n = {x | x 从某项起始终属于 A_n}`

若 `limsup A_n = liminf A_n = A`，则称 `A_n` 收敛到 `A`，记作 `lim A_n = A`。
