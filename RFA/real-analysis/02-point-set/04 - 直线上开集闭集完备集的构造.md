# 第四节 直线上开集、闭集、完备集的构造

这一节只做一件事：在实数轴 R 上，用最直观方式把开集、闭集、完备集（完全集）串成一条构造链。

## 一、先固定基础概念

1. 内点：存在一个邻域，整个都在集合里。  
2. 聚点：任意邻域里都有集合中的别的点。  
3. 开集：集合中每个点都是内点。

直接关系：
- 内点一定是聚点；
- 聚点不一定是内点。

## 二、从 (0,1) 开始看开集

设 E = (0,1)。

- 所有满足 0 < x < 1 的点都在 E 中，比如 0.1、0.0001、0.9999；
- 这些点都是内点，因此属于开集；
- 0 和 1 是聚点，但不是内点，不属于 E。

一句话：靠近端点但没碰到端点的点，仍属于开集。

## 三、直线上开集怎么构造

### 结论
R 上任意非空开集都可表示为：至多可列个、两两不相交开区间的并。

### 直观
直线上的开集本质就是“一堆拆开的开区间拼起来”，并且不包含各自端点。

## 四、直线上闭集怎么构造

### 结论
若 F 是闭集，则其补集 G = R \ F 是开集。  
反过来，闭集可以看成“从实数轴里挖去若干开区间后剩下的部分”。

### 直观
开集把边界聚点留在外面；闭集把这些聚点收回来。

例如：
- (0,1) 是开集；
- [0,1] 把边界聚点 0 和 1 收进来，是闭集。

## 五、直线上完备集（完全集）怎么构造

完备集（本章语境下即完全集）满足两个条件：
1. 是闭集（聚点不漏）
2. 没有孤立点（每个点都是聚点）

所以构造思路是：
- 先做成闭集；
- 再检查是否有孤立点，有就不完备。

例子：
- [0,1] 是完备集；
- 有限点集如 {1,2,3} 虽闭，但有孤立点，不是完备集。

## 六、一条逻辑链记住本节

1. 开集：只收内点，不收边界聚点  
2. 闭集：把边界聚点也收进来  
3. 完备集：在闭集基础上再要求“无孤立点”

## 七、考试速记

1. 开集：可列个不相交开区间之并  
2. 闭集：实数轴减去可列个开区间  
3. 完备集：闭集且无孤立点（等价 E = E'）
