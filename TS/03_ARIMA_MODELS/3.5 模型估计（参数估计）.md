# 3.5 模型估计（参数估计）

## 核心目标
估计 ARMA 模型中的系数 phi_1,...,phi_p、theta_1,...,theta_q 以及噪声方差 sigma_w^2（及可能包含的常数项，视设定而定）。

---

## 三种主流估计方法

### 1. 矩估计（Yule-Walker 等）
用样本自协方差匹配理论自协方差；对 **AR** 部分尤常用（尤尔–沃克方程）。

- 优点：计算快，小样本下 AR 部分较直观  
- 缺点：对一般 MA/ARMA 不如 MLE 精细

### 2. 条件最小二乘（CLS）
给定过去冲击的某种启动规则，最小化条件平方和。

- 优点：实现相对简单，适用范围较广  
- 缺点：对初值与截断规则敏感

### 3. 极大似然估计（MLE）
在（常见）高斯白噪声假设下最大化似然；statsmodels 默认路径多与此接近。

- 优点：大样本理论成熟，可输出标准误与信息准则  
- 缺点：依赖数值优化，可能陷入局部极值（实务需多给初值或网格）

---

## 模型定阶准则
对多组候选 (p,q) 拟合后，可用信息准则比较：

- **AIC**：AIC = -2 * ln(L) + 2 * k  
- **BIC**：BIC = -2 * ln(L) + k * ln(n)

其中 L 为极大似然值，k 为有效参数个数（含方差等），n 为样本长度。一般**越小越好**（在拟合与简洁之间折中）；实务中 BIC 往往更惩罚复杂模型。

---

## Python 示例
```python
from statsmodels.tsa.arima.model import ARIMA

model = ARIMA(x, order=(2, 0, 1))  # ARMA(2,1)
res = model.fit()
print(res.summary())  # 参数、标准误、AIC、BIC 等
```
