# 2.1 时间序列背景下的经典回归

## 核心目标
在时间序列分析的框架下，回顾和应用经典线性回归模型，理解其在序列数据中的适用性与局限性。

## 核心内容

### 1. 经典线性回归回顾
- **模型形式（纯文本）**  
  y_t = beta0 + beta1*x_{1t} + ... + betak*x_{kt} + epsilon_t  
  其中 y_t 为被解释变量，x_{1t},...,x_{kt} 为解释变量，beta0,...,betak 为待估系数，epsilon_t 为误差项。
- **基本假设**  
  误差项零均值、同方差、无自相关、与解释变量不相关（在经典横截面设定下常用；放到时序里后几条常需修正）。

### 2. 时间序列数据下的回归问题
- 解释变量与被解释变量之间的**时序相关**与**共同趋势**。
- 误差项的**序列自相关**（违背“无自相关”假设）带来的后果。
- **伪回归**：两个本无经济关系的非平稳序列因共同漂移而得到“显著”系数的现象，以及如何识别（单位根、协整等，后续章节会展开）。

### 3. 回归结果的统计推断
- 系数估计的性质与**标准误**计算；在存在自相关时，普通 OLS 标准误往往不可靠。
- 假设检验（t 检验、F 检验）在时序数据中的**修正思路**（如 Newey-West 异方差自相关稳健标准误等）。
- **拟合优度 R^2** 的解读与局限：时序里高 R^2 未必代表真实预测力或因果关系。

## 关键知识点
- 理解伪回归：两个不相关或关系很弱的非平稳序列做回归，可能得到看似显著、但无实际预测或因果意义的系数。
- 经典回归的“误差项无自相关”假设在时序数据中常不成立，会导致标准误偏误，进而影响显著性判断与置信区间。
