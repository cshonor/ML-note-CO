# 2.2 探索性数据分析（EDA）

## 核心目标
掌握用可视化与描述统计对时间序列做**体检**：识别趋势、季节、异常与分布特征，为建模与因子研究提供依据；明确 EDA 的定位是**理解数据**，而非一步定终模。

---

## 一、时序折线图是什么？

**时序折线图**（Time Series Line Plot）：把观测按**时间顺序**连成折线，横轴为时间，纵轴为变量取值。它是时序 EDA 的「第一道工序」，用来快速判断：

1. **趋势**：整体上升、下降还是大致水平  
2. **季节性 / 周期性**：是否每隔固定长度重复形态（如月度、周内效应）  
3. **异常点 / 突变**：尖峰、断崖、缺失导致的断线  
4. **波动与方差变化**：前后段波动尺度是否明显不同（异方差、波动聚集）

直观例子：

- **股价**：牛熊、趋势与黑天鹅导致的急跌  
- **销量**：大促周期性高峰与平时平缓段  

一句话：**先用眼睛搞清楚数据长什么样，再选模型与变换。**

### 1. 单序列折线图
- 横轴时间、纵轴观测，看**水平漂移、斜率趋势、周期起伏、尖峰与断点**  
- 对收益率、价差等，注意**波动聚集**

### 2. 多序列对比图
- 同图或多面板对比，看**同步、领先滞后、结构切换**  
- 为后续 CCF、回归、协整等提供直观假设

---

## 二、描述性统计

### 1. 位置与尺度
- 均值、中位数、标准差、分位数（含极值与四分位距）  
- 厚尾序列中均值易被极端点拉动，需结合中位数与稳健统计

### 2. 形状与离群
- **偏度、峰度**：相对正态的偏离（金融收益常尖峰厚尾）  
- **箱线图**：快速看离群点与对称性

---

## 三、分布与相关性探索

### 1. 直方图与 QQ 图
- 直方图看**单峰 / 多峰、偏斜**  
- QQ 图看**相对正态的偏离**（实务中多作参考，不必强行正态）

### 2. 散点图与相关矩阵
- 两两散点看**线性 / 非线性、杠杆点**  
- 相关矩阵给**线性相关**的全局快照（非线性会低估）

---

## 四、模式识别（从图到结论）

1. **趋势**：单调或缓慢漂移，提示差分或去趋势后再建模  
2. **季节 / 周期**：固定间隔重复，提示季节项、谐波或 SARIMA 等  
3. **突变与异常**：单点跳变或方差台阶，提示事件、数据错误或结构断点

---

## 五、完整 Python EDA 示例（可直接运行）

依赖：`numpy`、`pandas`、`matplotlib`、`scipy`（一般已随科学计算环境安装）。  
若需更美观的统计图，可自行安装 `seaborn` 并把下面作图换成 sns 接口。

无图形界面时可在仓库根目录运行：`python TS/02_REGRESSION_EDA/eda_ts_lineplot_demo.py`（与下面逻辑一致，使用 `Agg` 后端并把图保存到 `TS/02_REGRESSION_EDA/img/`）。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy import stats

# 中文显示：无中文字体时可注释掉下面两行，或改为本机已有字体
plt.rcParams["font.sans-serif"] = ["SimHei", "Microsoft YaHei", "DejaVu Sans"]
plt.rcParams["axes.unicode_minus"] = False

# ---------- 1. 模拟数据（可换成 pd.read_csv 读你的文件） ----------
np.random.seed(42)
n = 500
date_rng = pd.date_range(start="2020-01-01", periods=n, freq="D")

trend = 0.1 * np.arange(n)
seasonal = 5 * np.sin(2 * np.pi * np.arange(n) / 30)
noise = np.random.normal(0, 2, n)
ts1 = trend + seasonal + noise

ts2 = 0.8 * trend + 3 * np.cos(2 * np.pi * np.arange(n) / 30) + np.random.normal(0, 1.5, n)

df = pd.DataFrame({"date": date_rng, "value1": ts1, "value2": ts2})
df.set_index("date", inplace=True)

print("=== 数据前5行 ===")
print(df.head())

# ---------- 2. 时序折线图：单序列 + 多序列 ----------
plt.figure(figsize=(12, 8))

plt.subplot(2, 1, 1)
plt.plot(df["value1"], color="blue", linewidth=1.2, label="value1")
plt.title("单序列时序折线图", fontsize=14)
plt.ylabel("值")
plt.grid(alpha=0.3)
plt.legend()

plt.subplot(2, 1, 2)
plt.plot(df["value1"], color="blue", linewidth=1.2, label="value1")
plt.plot(df["value2"], color="orange", linewidth=1.2, label="value2")
plt.title("多序列对比时序折线图", fontsize=14)
plt.xlabel("时间")
plt.ylabel("值")
plt.grid(alpha=0.3)
plt.legend()

plt.tight_layout()
plt.show()

# ---------- 3. 描述性统计 ----------
print("\n=== 描述性统计 ===")
print(df.describe())

print("\n=== 偏度与峰度 ===")
print(f"value1 偏度: {df['value1'].skew():.3f}, 峰度: {df['value1'].kurtosis():.3f}")
print(f"value2 偏度: {df['value2'].skew():.3f}, 峰度: {df['value2'].kurtosis():.3f}")

# ---------- 4. 直方图 + QQ 图 ----------
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
plt.hist(df["value1"], bins=40, color="steelblue", edgecolor="white", alpha=0.85, density=True)
plt.title("value1 直方图（密度尺度）")
plt.xlabel("value1")

plt.subplot(1, 2, 2)
stats.probplot(df["value1"], plot=plt)
plt.title("value1 QQ 图")

plt.tight_layout()
plt.show()

# ---------- 5. 箱线图 ----------
plt.figure(figsize=(8, 4))
plt.boxplot([df["value1"], df["value2"]], tick_labels=["value1", "value2"])
plt.title("箱线图：离群值")
plt.grid(alpha=0.3)
plt.show()

# ---------- 6. 散点图 + 相关矩阵热力图 ----------
plt.figure(figsize=(10, 4))

plt.subplot(1, 2, 1)
plt.scatter(df["value1"], df["value2"], alpha=0.35, s=12)
plt.xlabel("value1")
plt.ylabel("value2")
plt.title("value1 vs value2 散点图")
plt.grid(alpha=0.3)

plt.subplot(1, 2, 2)
corr = df.corr()
im = plt.imshow(corr.values, cmap="coolwarm", vmin=-1, vmax=1)
plt.colorbar(im, fraction=0.046, pad=0.04)
plt.xticks(range(len(corr.columns)), corr.columns)
plt.yticks(range(len(corr.columns)), corr.columns)
for i in range(corr.shape[0]):
    for j in range(corr.shape[1]):
        plt.text(j, i, f"{corr.values[i, j]:.2f}", ha="center", va="center", color="black", fontsize=10)
plt.title("相关矩阵")

plt.tight_layout()
plt.show()
```

## 六、运行代码后你会看到什么？

1. **两张时序折线图**：单序列上看趋势 + 周期；双序列上看是否同向漂移、细节差异。  
2. **describe 与偏度、峰度**：分布位置、离散度、是否偏态与厚尾。  
3. **直方图 + QQ 图**：是否对称；QQ 尾部偏离直线常表示厚尾。  
4. **箱线图**：离群点大致位置与多变量对比。  
5. **散点图 + 热力图**：两变量线性相关强弱。

---

## 关键知识点总结
1. EDA 是建模与因子研究的**前置步骤**，重在提出假设、发现数据问题。  
2. **时序折线图**是信息密度最高的入口工具之一。  
3. 正态与线性相关在量化里多为**辅助**，结论需结合业务与稳健方法。
