# §1.1 随机事件及其运算

## 一、两类现象区分
### 1. 确定性现象
条件一旦固定，**结果唯一确定**，不存在偶然性。
- 例子：纯水标准大气压下100℃必然沸腾；苹果松手必然下落。

### 2. 随机现象
条件固定，但**结果不唯一**，单次结果无法预判；大量重复试验后会呈现稳定统计规律。
- 例子：抛硬币正反面、明天是否下雨、股票明日涨跌、掷骰子点数。

> 概率论专门研究：随机现象的统计规律性。

## 二、随机试验
满足三个核心特征，才能纳入概率研究范畴：
1. 试验可以**重复进行**
2. 试验所有可能结果**事先明确**，且不止一种
3. 试验结束前，**无法预知具体哪一个结果出现**

- 举例：掷骰子、抽签、放回式摸球，都是标准随机试验。

## 三、样本空间 Omega
### 1. 定义
随机试验**所有基本结果**构成的集合，记作 Omega；
其中每一个不可再拆分的最小结果，叫**样本点**。

### 2. 典型例子
- 抛1枚硬币：Omega = {正面, 反面}
- 掷1枚骰子：Omega = {1, 2, 3, 4, 5, 6}
- 连续抛2次硬币：Omega = {(正,正), (正,反), (反,正), (反,反)}

## 四、随机事件
### 1. 定义
样本空间 Omega 的**子集**，称为随机事件，简称事件；常用大写字母 A、B、C 表示。
通俗理解：我们关心的“某一种结果范围”。

### 2. 分类与特例
1. **基本事件**：只含**一个样本点**的事件
   - 例：掷骰子“点数为3”
2. **必然事件**：包含全部样本点，即 Omega，一定发生
   - 例：掷骰子“点数≤6”
3. **不可能事件**：空集 empty set，一定不会发生
   - 例：掷骰子“点数=7”

## 五、事件之间的关系
### 1. 包含关系 A 是 B 的子集（A subset B）
含义：A发生，则B一定发生。
- 例子：
  - A：掷骰子点数为2
  - B：掷骰子点数为偶数
  - 显然：A 是 B 的子集

### 2. 相等关系 A = B
A和B包含**完全相同的样本点**，本质是同一个事件。

### 3. 互斥关系（互不相容）
A 交 B = empty set
含义：A、B**不可能同时发生**。
- 例子：抛硬币“正面”和“反面”互斥；
- 注意：可以都不发生，只是不能同时发生。

### 4. 对立事件（逆事件）A 的对立事件（A bar）
满足两个条件：
1. A 并 A 的对立 = Omega（必有一个发生）
2. A 交 A 的对立 = empty set（不能同时发生）

- 例子：
  - A：掷骰子点数为偶数
  - A 的对立：掷骰子点数为奇数

> 对立 一定 互斥；互斥 不一定 对立。

## 六、事件的运算
设 A、B 为任意事件

### 1. 并事件 A 并 B
含义：**A发生 或 B发生（至少一个）**
口语：A或B

### 2. 交事件 A 交 B（简写 AB）
含义：**A、B同时发生**
口语：A且B

### 3. 差事件 A - B
含义：**A发生，且B不发生**
等价写法：A - B = A 与 非B 同时发生

## 七、事件运算定律
1. 交换律
A 并 B = B 并 A；A 交 B = B 交 A

2. 结合律
(A 并 B) 并 C = A 并 (B 并 C)
(A 交 B) 交 C = A 交 (B 交 C)

3. 分配律
A 交 (B 并 C) = (A 交 B) 并 (A 交 C)
A 并 (B 交 C) = (A 并 B) 交 (A 并 C)

4. 德摩根律（高频必考）
非(A 并 B) = 非A 交 非B
非(A 交 B) = 非A 并 非B

- 白话记忆：
  并的对立 = 对立的交
  交的对立 = 对立的并
