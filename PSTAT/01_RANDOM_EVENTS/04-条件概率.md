# §1.4 条件概率

## 一、条件概率核心定义
设 A、B 为两个事件，且 P(B)>0，则**在B已经发生的前提下，A发生的条件概率**：

P(A|B) = P(AB) / P(B)

### 直观理解
原本样本空间是全集 Omega；
现在**已知B发生**，相当于把样本空间**缩小到B范围内**，再看A在B里占比多少。

### 简单例子
掷骰子：
- B：点数为偶数 {2,4,6}
- A：点数为2

求：已知是偶数，恰好是2的概率
P(A|B) = (1/6) / (3/6) = 1/3

## 二、乘法公式
由条件概率变形直接得到：
P(AB) = P(B) * P(A|B) = P(A) * P(B|A)

### 意义
用来计算**两个事件同时发生**的概率。

### 推广：链式乘法（多事件）
P(A1A2A3) = P(A1) * P(A2|A1) * P(A3|A1A2)
常用于不放回抽样、连续分步概率问题。

## 三、全概率公式（由因推果）
### 1. 完备事件组划分
B1, B2, ..., Bn 满足：
1. 两两互斥
2. B1并B2并...并Bn = Omega

相当于把整个样本空间**不重不漏切分**。

### 2. 全概率公式
P(A) = P(B1)P(A|B1) + P(B2)P(A|B2) + ... + P(Bn)P(A|Bn)

### 白话理解
A可以在各种不同原因/场景 Bi 下发生；
把每一种场景下A的概率，按场景权重加总，就是A总的概率。

### 经典场景
多个箱子装球、多渠道进货、多种病因导致同一症状，都用全概率。

## 四、贝叶斯公式（由果溯因，核心灵魂）

### 1. 公式定义（完备事件组形式）
设 B1, B2, …, Bn 是样本空间的**互斥且完备**的原因组（即 B1 并 … 并 Bn = Omega，且 Bi 与 Bj 两两不交），且 P(Bi) > 0；A 为任意结果事件（P(A) > 0）。则：

P(Bi|A) = P(Bi) P(A|Bi) / [ P(B1)P(A|B1) + P(B2)P(A|B2) + … + P(Bn)P(A|Bn) ]

分母即对 j 从 1 到 n 求和：sum_j P(Bj) P(A|Bj)。

- **分子**：P(Bi)（**先验概率**：观察结果前，原因 Bi 发生的初始概率）× P(A|Bi)（**似然度**：原因 Bi 发生时，结果 A 出现的概率）。
- **分母**：P(A)（**全概率**：结果 A 发生的总概率，由全概率公式计算）。
- **左边**：P(Bi|A)（**后验概率**：观察到结果 A 后，原因 Bi 发生的更新概率）。

### 2. 核心逻辑：由果溯因（逆向推理）
- **全概率公式**：由因推果——已知各原因概率，求结果发生的总概率。
- **贝叶斯公式**：由果溯因——已知结果已发生，反推是由某一原因 Bi 导致的概率。
- 一句话：**用结果修正对原因的判断**，把先验概率 P(Bi) 更新为后验概率 P(Bi|A)。

### 3. 公式推导（极简）
从**条件概率定义**出发：P(Bi|A) = P(BiA) / P(A)。

- **分子**用**乘法公式**：P(BiA) = P(Bi) P(A|Bi)。
- **分母**用**全概率公式**：P(A) = sum_j P(Bj) P(A|Bj)。

代入即得贝叶斯公式。

### 4. 通俗例子（快速理解）
**问题**：某工厂有 3 条生产线（B1, B2, B3），产量占比分别为 50%、30%、20%；次品率分别为 1%、2%、3%。现抽到 1 件次品（结果 A），求它来自 B1 的概率 P(B1|A)。

**解**

1. 先验概率：P(B1)=0.5，P(B2)=0.3，P(B3)=0.2。  
2. 似然度：P(A|B1)=0.01，P(A|B2)=0.02，P(A|B3)=0.03。  
3. 全概率（分母）：P(A) = 0.5×0.01 + 0.3×0.02 + 0.2×0.03 = 0.017。  
4. 后验概率：P(B1|A) = (0.5×0.01) / 0.017 ≈ 0.294 ≈ 29.4%。

**结论**：抽到次品后，它来自 B1 的概率从先验 50% 修正为后验约 29.4%。

### 5. 关键术语对照

| 符号 | 名称 | 含义 |
| :--- | :--- | :--- |
| P(Bi) | 先验概率 | 观察结果前，原因 Bi 的初始概率 |
| P(A|Bi) | 似然度 | 原因 Bi 下，结果 A 出现的概率 |
| P(A) | 证据（全概率） | 结果 A 发生的总概率 |
| P(Bi|A) | 后验概率 | 观察结果后，原因 Bi 的更新概率 |

### 6. 一句话总结
贝叶斯公式 = **条件概率 + 全概率**，核心是**用结果反向修正原因概率**，实现由果溯因的推理，是统计推断、机器学习等的基础工具之一。

### 应用场景
疾病诊断、垃圾邮件分类、金融风险归因、贝叶斯推断、分类模型底层思想。
