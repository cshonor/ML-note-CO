# §1.4 条件概率

## 一、条件概率核心定义
设 A、B 为两个事件，且 P(B)>0，则**在B已经发生的前提下，A发生的条件概率**：

P(A|B) = P(AB) / P(B)

### 直观理解
原本样本空间是全集 Omega；
现在**已知B发生**，相当于把样本空间**缩小到B范围内**，再看A在B里占比多少。

### 简单例子
掷骰子：
- B：点数为偶数 {2,4,6}
- A：点数为2

求：已知是偶数，恰好是2的概率
P(A|B) = (1/6) / (3/6) = 1/3

## 二、乘法公式
由条件概率变形直接得到：
P(AB) = P(B) * P(A|B) = P(A) * P(B|A)

### 意义
用来计算**两个事件同时发生**的概率。

### 推广：链式乘法（多事件）
P(A1A2A3) = P(A1) * P(A2|A1) * P(A3|A1A2)
常用于不放回抽样、连续分步概率问题。

## 三、全概率公式（由因推果）
### 1. 完备事件组划分
B1, B2, ..., Bn 满足：
1. 两两互斥
2. B1并B2并...并Bn = Omega

相当于把整个样本空间**不重不漏切分**。

### 2. 全概率公式
P(A) = P(B1)P(A|B1) + P(B2)P(A|B2) + ... + P(Bn)P(A|Bn)

### 白话理解
A可以在各种不同原因/场景 Bi 下发生；
把每一种场景下A的概率，按场景权重加总，就是A总的概率。

### 经典场景
多个箱子装球、多渠道进货、多种病因导致同一症状，都用全概率。

## 四、贝叶斯公式（由果溯因，核心灵魂）
在全概率基础上，已知结果A发生，反推是由某一个原因 Bi 导致的概率：

P(Bi|A) = P(Bi)P(A|Bi) / [P(B1)P(A|B1)+...+P(Bn)P(A|Bn)]

### 名词解释
- P(Bi)：**先验概率**——事情发生前，对原因的初始判断
- P(Bi|A)：**后验概率**——看到结果后，修正后的原因概率

### 应用场景
疾病诊断、垃圾邮件分类、金融风险归因、贝叶斯推断、AI分类底层思想。
