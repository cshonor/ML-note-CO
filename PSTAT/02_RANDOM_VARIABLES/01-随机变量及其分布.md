# §2.1 随机变量及其分布

## 2.1.1 随机变量的概念
- 定义：定义在样本空间 Omega 上的实值函数 X = X(w)，其中 w 属于 Omega。
- 本质：把随机试验的结果“数值化”，让事件变成可计算的数。
- 例子：抛硬币，定义 X=1（正面），X=0（反面），则 X 就是一个随机变量。

## 2.1.2 随机变量的分布函数
- 定义：F(x) = P(X<=x)，其中 x 属于实数集 R。
- 性质：
  1. 单调不减
  2. 右连续
  3. F(-∞)=0，F(+∞)=1
- 用途：描述随机变量的整体概率分布情况。

### 与「黎曼积分」的关系（易混点）
- **定义 F(x) 不需要积分**：F(x) 是事件 {X<=x} 的概率，由概率公理与 X 的分布完全决定；离散型用**求和**即可写出 F（阶梯函数）。
- **积分出现在连续型**：若存在密度 f(x)，使得 F(x) 等于从 -∞ 到 x 对 f(t) 的积分，本科教材里常把这个积分按**黎曼意义下的反常积分**来算（分段光滑、常见分布都够用）。
- **更一般的理论**：概率论里更自然的是**勒贝格积分**及对分布函数 dF 的**斯蒂尔切斯积分**（统一离散求和与连续积分）；初学可先记：**F 是核心，积分只是连续情形下的计算工具**。

## 2.1.3 离散型随机变量的概率分布列

### 定义
若随机变量 X 至多取可列个值 x1, x2, …，且

P(X = x_k) = p_k，k = 1, 2, …

则称 {p_k} 为 X 的**概率分布列**（简称分布列），须满足：

1. p_k >= 0（非负）  
2. 对所有 k 求和：sum_k p_k = 1（全部可能结果的概率之和为 1）

### 与分布函数的关系（离散型）
F(x) = P(X <= x) 等于「所有满足 x_k <= x 的 p_k 之和」，即

F(x) = sum_{x_k <= x} p_k

故离散型的 F(x) 是**阶梯函数**，在 x_k 处有跳跃，跃度为 p_k。

### 例子

**例 1（均匀硬币）** 抛一枚均匀硬币，令 X=0 表示反面、X=1 表示正面，则分布列为：

| X | 0 | 1 |
| :---: | :---: | :---: |
| P | 0.5 | 0.5 |

**例 2（均匀骰子）** 掷一枚均匀骰子，点数 X 的分布列为：

| X | 1 | 2 | 3 | 4 | 5 | 6 |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| P | 1/6 | 1/6 | 1/6 | 1/6 | 1/6 | 1/6 |

**例 3（两点分布，一般形式）** 若 P(X=1)=p，P(X=0)=1-p，其中 0<p<1，则仍满足非负且和为 1；例 1 即 p=0.5 的特例。更一般的离散分布见本章「常用离散分布」一节。

## 2.1.4 连续型随机变量的概率密度函数
- 定义：存在非负函数 f(x)，使得 F(x)=从 -∞ 到 x 的 f(t) 对 t 的积分，则 X 为连续型随机变量，f(x) 为其概率密度函数。
- 性质：
  1. f(x) >= 0
  2. 从 -∞ 到 +∞ 的 f(x) 积分等于 1
- 注意：连续型随机变量在单点的概率为 P(X=a)=0。

### 区间上的概率（由 F 或 f 算）
对任意 a < b，有 P(a < X <= b) = F(b) - F(a) = 从 a 到 b 对 f(x) 的积分（与是否含端点 a、b 无关，因单点概率为 0）。

### 例子

**例 1（区间 [a,b] 上的均匀分布，a<b）** 令

- 当 a < x < b 时，f(x) = 1 / (b-a)；
- 其余 x 处 f(x) = 0。

则从 -∞ 到 +∞ 对 f 积分 = (b-a) * 1/(b-a) = 1；当 a <= x <= b 时，F(x) = 从 a 到 x 积分 = (x-a)/(b-a)。例如在 [0,1] 上均匀时，0<x<1 内 f(x)=1，F(x)=x。

**例 2（指数分布，参数 lambda>0）** 令

- 当 x > 0 时，f(x) = lambda * e^(-lambda x)（e 为自然对数的底）；
- 当 x <= 0 时，f(x) = 0。

可验证从 0 到 +∞ 对 f(x) 积分 = 1；且 F(x) = 0（若 x<=0），F(x) = 1 - e^(-lambda x)（若 x>0）。

**例 3（正态分布）** 正态密度含系数 1/(sigma*sqrt(2pi)) 与指数项 e^(-(x-mu)^2/(2 sigma^2))，全直线积分为 1；具体形式与图像见本章「常用连续分布」一节。
