# 第五节 康托尔三分集

这一节完全承接第四节：在一维直线上，把开集、闭集、完备集的逻辑放到一个经典构造里。

## 一、先回忆本章主线

1. 开集：只有内点，不含边界聚点  
2. 闭集：包含自身全部聚点  
3. 完备集（完全集）：闭集且没有孤立点（每个点都是聚点）

康托尔集的关键价值就是：  
看起来像“被挖空”，却仍是完备集。

## 二、康托尔集怎么构造

从区间 [0,1] 开始：

1. 第一次去掉中间开区间 (1/3, 2/3)，剩下  
   [0,1/3] 并 [2/3,1]
2. 第二次对每段再去掉中间三分之一开区间
3. 之后无限重复：每一步都“挖中间开区间”，并保留端点

所有步骤做完后剩下的点集，记作 C（康托尔三分集）。

## 三、为什么它是闭集

每一步挖掉的是开区间。  
“挖掉部分”的总并是开集。  
而 C = [0,1] 减去这个开集，所以 C 是闭集。

## 四、为什么它是完备集

完备集要满足两条：
1. 闭集（已满足）
2. 没有孤立点

对 C 中任一点 x，不管邻域多小，附近都还能找到 C 中其他点。  
因此 C 中每点都是聚点，没有孤立点。  
所以 C 是完备集（完全集）。

## 五、最反直觉的三个结论

1. C 是完备集（闭且无孤立点）  
2. C 的勒贝格测度是 0（被挖去长度总和为 1）  
3. C 仍是不可数集（点的多少与 [0,1] 同等级）

## 六、和第四节一条线串起来

1. 先学内点、聚点  
2. 再学开集（只收内点）  
3. 再学闭集（收全聚点）  
4. 再学完备集（闭且无孤立点）  
5. 康托尔集把这四步同时体现出来

## 七、考试速记

- 构造：从 [0,1] 起，不断去掉每段中间三分之一开区间  
- 性质：闭、无内点、无孤立点、测度 0、不可数  
- 定位：一维直线上“完备集但无区间”的典型反直觉例子
