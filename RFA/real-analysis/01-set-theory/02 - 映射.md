# 实变函数 第一章 第二节：映射

## 一、映射定义

设 `X, Y` 为非空集合。若存在对应法则 `f`，使得对任意 `x ∈ X`，都有唯一 `y ∈ Y` 与之对应，称 `f` 为从 `X` 到 `Y` 的映射，记作：

`f: X -> Y`

其中：

- `X` 是定义域 `D(f)`
- `Y` 是陪域
- `f(X) = {f(x) | x ∈ X}` 是值域 `R(f)`

## 二、映射类型

- 单射（injective）：`x1 != x2 => f(x1) != f(x2)`
- 满射（surjective）：`f(X) = Y`
- 双射（bijective）：既单又满

双射是后续“集合对等”的核心。

## 三、复合映射

设 `f: X -> Y`，`g: Y -> Z`，定义：

`(g o f)(x) = g(f(x))`

注意复合顺序不能随意交换。

## 四、逆映射

若 `f: X -> Y` 是双射，则存在唯一映射 `f^(-1): Y -> X`，满足：

- `f^(-1)(f(x)) = x`
- `f(f^(-1)(y)) = y`

结论：只有双射才有逆映射。

## 五、像与原像

### 1. 像

若 `A ⊂ X`，则

`f(A) = {f(x) | x ∈ A}`

### 2. 原像

若 `B ⊂ Y`，则

`f^(-1)(B) = {x ∈ X | f(x) ∈ B}`

### 3. 原像运算律（常考）

- `f^(-1)(∪ B_alpha) = ∪ f^(-1)(B_alpha)`
- `f^(-1)(∩ B_alpha) = ∩ f^(-1)(B_alpha)`
- `f^(-1)(B \ C) = f^(-1)(B) \ f^(-1)(C)`
