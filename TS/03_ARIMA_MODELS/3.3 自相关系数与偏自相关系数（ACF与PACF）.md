# 3.3 自相关系数与偏自相关系数（ACF 与 PACF）

## 核心概念
- **ACF（自相关函数）**：序列与自身滞后 k 期的线性相关强度（标准化后多在 -1 到 1）
- **PACF（偏自相关函数）**：控制中间滞后 1,...,k-1 之后，序列与滞后 k 期的**直接**相关强度

---

## 三种模型的判断规则（重点）
| 模型 | ACF | PACF |
| :--- | :--- | :--- |
| AR(p) | 拖尾（指数或振荡衰减） | 截尾（p 阶后落入置信区间） |
| MA(q) | 截尾（q 阶后落入置信区间） | 拖尾（指数或振荡衰减） |
| ARMA(p,q) | 拖尾 | 拖尾 |

- **截尾**：若干阶之后系数快速落入置信区间，统计上不显著
- **拖尾**：系数缓慢衰减，长期有小幅起伏

---

## 用 Python 画图（statsmodels）
```python
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf
import matplotlib.pyplot as plt

# 假设 x 为已处理成近似平稳的一维序列
fig, axes = plt.subplots(2, 1, figsize=(8, 6))
plot_acf(x, lags=20, ax=axes[0])
plot_pacf(x, lags=20, method="ywm", ax=axes[1])
plt.tight_layout()
plt.show()
```

---

## 应用：模型识别（口诀）
- PACF 在 p 阶附近截尾，可优先考虑 **AR(p)**
- ACF 在 q 阶附近截尾，可优先考虑 **MA(q)**
- 两者都拖尾，考虑 **ARMA(p,q)**，并结合 **AIC/BIC** 与残差诊断定阶
