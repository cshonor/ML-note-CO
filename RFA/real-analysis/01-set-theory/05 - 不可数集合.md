# 实变函数 第一章 第五节：不可数集合

## 一、定义

不可数集合指：

- 是无限集合
- 且不能与 `N` 对等

即“无法编号成一列”的无限集合。

## 二、典型例子

- 区间 `(0,1)`
- 实数集 `R`

`(0,1)` 不可数通常用康托尔对角线法证明。

## 三、康托尔对角线法（思路）

反证：假设 `(0,1)` 可列成序列。  
把每个数写成小数形式，沿对角线逐位改造，构造出一个新数，使其第 `n` 位与第 `n` 个数第 `n` 位不同。  
该新数不在原序列中，矛盾，故不可数。

## 四、连续统基数

`R` 的基数记作连续统基数 `c`（有时记作 `aleph`），并有：

`aleph_0 < c`

即可数无穷严格小于实数不可数无穷。

## 五、常见结论

- 各类实区间（开、闭、半开）基数相同，均为 `c`
- 不可数集去掉可数子集后仍可能不可数（实分析中常见结论）
- 复数平面点集与 `R` 在基数上同阶（与连续统同级）

## 六、和可数集的本质区别

- 可数集：可排成序列，可逐个编号
- 不可数集：无法排成序列，规模严格更大
