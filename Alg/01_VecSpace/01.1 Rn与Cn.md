# 1.A R^n 与 C^n

## 1) 直观对象

- R^n：n 维实向量空间，元素是 n 维实数组  
  x = (x1, x2, ..., xn), 每个 xi 属于 R
- C^n：n 维复向量空间，元素分量属于 C

在应用中可理解为：

- 因子暴露向量
- 收益向量
- 风险参数向量

## 2) 两个基本运算

1. 向量加法（分量对应相加）  
   (x1,...,xn) + (y1,...,yn) = (x1+y1,...,xn+yn)
2. 数乘（标量乘每个分量）  
   lambda*(x1,...,xn) = (lambda*x1,...,lambda*xn)

## 3) 八条基础规则（后面变成公理）

对任意向量 u,v,w 和标量 lambda,mu：

1. u+v = v+u
2. (u+v)+w = u+(v+w)
3. 存在 0，使 v+0 = v
4. 对每个 v 存在 -v，使 v+(-v)=0
5. lambda*(u+v) = lambda*u + lambda*v
6. (lambda+mu)*v = lambda*v + mu*v
7. lambda*(mu*v) = (lambda*mu)*v
8. 1*v = v
