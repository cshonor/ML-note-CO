# 线性代数（Axler 主线）
教材主线：《线性代数应该这样学》（Axler）

归属仓库：机器学习仓库  
配套学习：凸优化、实变函数、机器学习、常微分方程

本目录按 Axler 的现代线性代数逻辑组织：  
先向量空间与线性映射，再特征值与内积空间算子，最后学习迹与行列式。

## 为什么采用这条主线
- 行列式放在后段处理，不再作为入门核心障碍。
- 以线性映射（算子）为主角，矩阵只是基下表示。
- 结构直接服务高维几何、优化建模、量化研究与机器学习。
- 谱定理、SVD、正规/自伴算子等内容处于主线核心位置。

## 章节学习顺序（Axler 对齐）
### 第0部分 前置热身（本仓库补充）
- `00_Preliminary_LinSys`：线性方程组、消元法、数域基础直观

### 第1章 向量空间
- `01_VecSpace`：子空间、直和、线性无关、张成与结构直觉

### 第2章 有限维向量空间
- `02_FiniteDim`：基、维数、坐标表示与有限维结构

### 第3章 线性映射
- `03_LinearMap`：核与像、矩阵表示、同构、积与商、对偶

### 第4章 多项式
- `04_Polynomial`：多项式代数、最小多项式、算子多项式工具

### 第5章 特征值与不变子空间
- `05_Eigen_Subspace`：特征值、特征向量、不变子空间、三角化主线

### 第6章 内积空间
- `06_InnerProductSpace`：内积、范数、正交、投影、最小二乘

### 第7章 内积空间上的算子
- `07_InnerProduct_Operator`：伴随、自伴、正规、酉/正交算子

### 第8章 复空间算子
- `08_Complex_Operator`：复内积空间上的谱结构与标准形

### 第9章 实空间算子
- `09_Real_Operator`：实空间情形下的结构结论与分解

### 第10章 迹与行列式（后置）
- `10_Trace_Determinant`：迹、行列式及其算子意义（非入门前置）

### 第11部分 二次型（本仓库补充）
- `11_QuadraticForm`：二次型、正定性、优化中的曲率理解

### 第12部分 矩阵计算（本仓库补充）
- `12_Matrix_Computation`：LU/QR/SVD 的数值计算视角与工程实现

## 量化路线学习建议
1. 主线严格跟 Axler：先空间与算子，再谱结构与分解。
2. 并行补充 Strang：加强数值计算直觉与工程实现视角。
3. 传统教材定位为习题库：哪里计算不熟，就回刷对应题组。
4. 二次型与矩阵计算放在后段，避免打断抽象主线。
