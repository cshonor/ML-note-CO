# 第 1 章笔记：什么是机器学习

## 本章笔记整理

### 精简版

- 人类决策有两类：规则推理、经验判断。  
- 机器学习对应“经验判断”，其中经验在计算机里就是数据。  
- 当计算机通过数据学习并决策时，就是在进行机器学习。  
- 对复杂任务（如图像识别），硬编码规则难度高，数据驱动方法更有效。  

### 详细版

#### 核心概念

- **经验 = 数据**：机器学习不是人工穷举规则，而是让模型从数据中归纳规律。  
- **范式变化**：从“程序 + 数据 -> 结果”逐步转向“数据 + 学习算法 -> 模型 -> 结果”。  

#### 与传统编程对比

- 传统编程：人先定义规则，机器执行规则。  
- 机器学习：人提供样本，模型自动学习决策边界。  
- 当任务特征难以完全形式化时（如视觉、语音、自然语言），机器学习通常更可行。  

#### 示例理解（苹果识别）

- 手写规则法：需定义颜色、形状、大小、纹理、光照变化等大量条件，容易遗漏。  
- 数据学习法：通过大量标注样本学习“苹果”特征分布，泛化到新样本。  

#### 一句话总结

> 机器学习的本质是让计算机从数据中积累经验，并用这些经验进行预测和决策。
