# 2.2 探索性数据分析（EDA）

## 核心目标
学习如何通过可视化与统计方法，对时间序列数据进行初步探索，发现数据中的模式、趋势、异常值与分布特征。

## 核心内容

### 1. 时序图分析
- 绘制序列随时间变化的折线图，识别趋势、季节性、周期和异常点。
- 多序列对比图，观察变量间的同步或领先滞后关系。

### 2. 描述性统计
- 均值、方差、偏度、峰度等统计量的计算与解读。
- 分位数、箱线图，识别数据分布与离群值。

### 3. 分布与相关性探索
- 直方图、QQ 图，用于直观检验序列是否近似正态（注意：金融收益常呈厚尾，正态只是参考基准）。
- 散点图与相关矩阵，探索变量间的线性关系强度与方向。

### 4. 模式识别
- 趋势的初步判断（上升 / 下降 / 无趋势）。
- 季节性或周期性的视觉识别。
- 突变点、异常值的定位与初步分析。

## 关键知识点
- 探索性数据分析是建模前的关键步骤，目的是“理解数据”，而不是直接给出最终模型。
- 时序图是最基础也最重要的工具，大部分序列特征都可以通过它先做一轮判断。
