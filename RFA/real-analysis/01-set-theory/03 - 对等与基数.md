# 实变函数 第一章 第三节：集合的对等与基数

## 一、集合对等（等势）

若存在双射 `f: A -> B`，则称 `A` 与 `B` 对等，记作：

`A ~ B`

直观：`A` 与 `B` 可以一一配对，无遗漏、无重复。

## 二、对等关系性质

`~` 是等价关系：

- 自反性：`A ~ A`
- 对称性：`A ~ B => B ~ A`
- 传递性：`A ~ B` 且 `B ~ C => A ~ C`

## 三、有限与无限的关键区别

- 有限集合不能与自身真子集对等
- 无限集合可与自身某个真子集对等（无限本质特征）

经典例子：

- `N ~ {2,4,6,...}`，可用 `f(n)=2n`
- `(0,1) ~ R`，可用反正切型双射构造

## 四、伯恩斯坦定理（重点）

若 `A` 与 `B` 的某子集对等，且 `B` 与 `A` 的某子集对等，则：

`A ~ B`

作用：很多场景不直接构造双射，也能判定对等。

## 五、基数（势）

若 `A ~ B`，称 `A` 与 `B` 基数相等。

基数用于描述集合“大小”，是有限个数概念对无限集合的推广。

### 基数比较

- `|A| <= |B|`：`A` 与 `B` 的某子集对等
- `|A| < |B|`：`|A| <= |B|` 且 `A` 不与 `B` 对等

### 重要结论

- 任意集合 `A` 都有 `|A| < |P(A)|`（幂集更大）
- 因此不存在“最大基数”
