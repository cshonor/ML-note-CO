# 第 3 章笔记：线性回归

## 本节核心

- 线性回归是一种常用的数值预测方法，用来估计连续值。  
- 典型应用：房价、股票价值、寿命、用户观看时长等。  
- 目标是找到一条“尽量贴近所有数据点”的直线。  

## 直观理解

- 图中散点代表观测数据，整体呈现近似线性趋势。  
- 我们希望画出一条线，让它与这些点的整体距离最小。  
- 这条线就是线性回归模型对数据规律的近似表达。  

## 类比（书中思路）

- 把散点看成城市中的房屋位置。  
- 画出的回归线像一条穿过城市的道路。  
- 道路越靠近大多数房屋，说明这条线越“符合需求”。  

## 一句话总结

> 线性回归就是用一条直线去近似数据中的变化趋势，并据此做连续值预测。
