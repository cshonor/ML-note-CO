# §3.2 边际分布与随机变量的独立性

## 1. 边际分布函数
F_X(x)=F(x,+∞)，F_Y(y)=F(+∞,y)

## 2. 离散型边际分布
p_i. = sum_j p_ij，p_.j = sum_i p_ij

## 3. 连续型边际密度
f_X(x) = 对 y 从 -∞ 到 +∞ 的积分 f(x,y)dy

f_Y(y) = 对 x 从 -∞ 到 +∞ 的积分 f(x,y)dx

## 4. 独立性定义
F(x,y)=F_X(x)F_Y(y)

- 离散：p_ij = p_i. * p_.j
- 连续：f(x,y)=f_X(x)f_Y(y)

## 5. 重要结论
独立 => 不相关；不相关 =/> 独立。

二维正态中：独立 当且仅当 rho=0。
